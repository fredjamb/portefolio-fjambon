PROJET LAFLEUR V2

Version 2 (V2)

L'ensemble du catalogue est répertorié dans une base de données MySql : il suffirait que les informations de cette base soient publiées sur le Web !

Il s'agissait de concevoir un site en langage PHP faisant appel à une base de données grâce à des requêtes SQL.

Cette base de données doit être créée à l’aide de « MySql » et sera utilisée dans les scripts PHP.

La nouvelle version du site sera composée de :

  • • Pages HTML stockées dans le répertoire « /Lafleur/Version2».
  • • Images stockées dans le répertoire « /Lafleur/Images ».
  • • Documents contenant des scripts PHP stockés dans le répertoire « /Lafleur/Version2».

Page d'accueil

La page d'accueil ne change quasiment pas. Seule la référence à menu.htm doit être remplacée par une référence au document PHP : « menu.php ». La page logo.htm n'est pas modifiée.
